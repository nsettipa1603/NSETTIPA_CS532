Lab-3

Program:
C program that implements two filenames as command-line arguments and 
concatenates the contents of the second file to the first file without changing the contents in the second file.

// file1.txt:-
//This is Neelima
// file2.txt:- 
//This is Settipalli

Steps for compilation:
cat file1.txt
cat file2.txt
'gcc -Wall nsettipa_lab3 -o nsettipa_lab3.c'


Steps for Execution:
./nsettipa_lab3 file1.txt file2.txt

// Output in file1:-
This is Neelima
This is Settipalli
//Output in file2:-
This is Settipalli
